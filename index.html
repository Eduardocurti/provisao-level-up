⚙️ Fórmula de Cálculo da Provisão Mensal
PJ — Múltiplo Salarial
PLR Anual = Sal. × Múltiplo
Prov. Mês = PLR × Avos ÷ 12
Ex: Gerente R$20k → PLR anual R$80k → provisão mês 1 = R$6.666,67
CLT — Valor Fixo
PLR Anual = Valor Fixo
Prov. Mês = Fixo × Avos ÷ 12
Ex: Analista Pleno → PLR R$4k → provisão mês 1 = R$333,33
Gatilho (EBITDA)
Pagamento Final =
PLR × Fator EBITDA
O EBITDA é o gate global — se <85%, nenhum pagamento é realizado independente das metas individuais.

⚠️ A forma recomendada de atualizar é sempre pela aba ⬆ Importar Mês. O guia manual abaixo serve apenas como referência ou para situações excepcionais.

Fluxo Mensal — Como Atualizar o Sistema

1

Use a aba ⬆ Importar Mês — forma recomendada

Todo mês, acesse a aba Importar Mês na navegação superior. O fluxo completo é:

1. Salve a planilha do mês no mesmo formato dos meses anteriores (.xlsx) 2. Acesse a aba "⬆ Importar Mês" 3. Arraste ou clique para selecionar o arquivo 4. O sistema detecta a aba automaticamente (ex: ABR26, Mai26...) 5. Confirme o mês de referência e clique em "Processar Importação" 6. Revise as alterações detectadas (entradas, saídas, reajustes) 7. Clique em "Confirmar e Atualizar Sistema" 8. O dashboard é atualizado automaticamente — sem necessidade de baixar novo arquivo
2

Nome da aba no arquivo .xlsx

O sistema reconhece automaticamente qualquer variação do nome do mês. Exemplos aceitos:

ABR26, Abr26, abr26 → detectado como Abril/2026 Mai26, MAI26, Maio26 → detectado como Maio/2026 Jun26, Junho26 → detectado como Junho/2026 // O sistema ignora maiúsculas/minúsculas e variações de escrita
3

O que o sistema detecta automaticamente

Ao comparar o arquivo importado com a base do mês anterior, o sistema identifica:

ENTRADA CNPJ que não existia no mês anterior → novo prestador SAÍDA CNPJ que estava no mês anterior e não aparece mais → desligamento REAJUSTE Mesmo CNPJ com valor de remuneração diferente → reajuste salarial
4

Regras PLR — como o cargo determina o cálculo

O sistema lê o campo Desc. Função da planilha e aplica a regra automaticamente:

ANALISTA + PLENO → R$ 4.000 fixo anual (independe do salário) ANALISTA + SENIOR → R$ 6.000 fixo anual (independe do salário) ESPECIALISTA → 2× salário mensal COORDENADOR/A → 2× salário mensal GERENTE → 4× salário mensal DIRETOR/A → 6× salário mensal CFO → 6× salário mensal
5

Como a provisão é calculada

A lógica de cálculo segue o critério de 1 avo por mês:

// Provisão do mês = 1 avo (PLR anual ÷ 12) Gerente R$ 20.000 → PLR anual = R$ 80.000 → provisão/mês = R$ 6.666,67 // Provisão acumulada YTD = soma dos avos mensais Jan: R$ 6.666,67 | Fev: + R$ 6.666,67 | Mar: + R$ 6.666,67 YTD Mar = R$ 20.000,00 (3 avos acumulados) // Gate do EBITDA é aplicado no pagamento final, não na provisão
6

Cenários 100% e 50% de Meta

Os dois cenários ficam sempre disponíveis no dashboard. O sistema provisiona com base em 100% da meta como referência máxima. O cenário 50% mostra metade dos valores para planejamento conservador. O Gatilho (EBITDA) é aplicado no pagamento final — não impacta a provisão mensal.
